使用 Hexo 生成静态文件快速而且简单。

监视文件变动

Hexo 能够监视文件变动并立即重新生成静态文件，在生成时会比对文件的 SHA1 checksum，只有变动的文件才会写入。

代码高亮设置

3个反斜杠加想要设置的变成语言,比如【3反斜杠+java 代码体 3反斜杠结束】就呈现效果如下所示：

分类设置

创建

hexo new page categories

目录引用

到生成的categories／index.md目录设置type类型指向。

路径配置

到你对应的主题（next）配置目录路径。

小图标设置

找到对应配置文件（next/layout/_partials/head.swig） 修改 href 地址引用即可。

标题位置设置

margin设置上下位置text-align 设置左右居中.

.posts-expand .post-meta {
  margin: 3px 0 20px 0;
  color: #999;
  font-family: 'Lato', "PingFang SC", "Microsoft YaHei", sans-serif;
  font-size: 12px;
  text-align: left;
}

.posts-expand .post-meta {
  margin: 3px 0 60px 0;
  color: $grey-dark;
  font-family: $font-family-posts;
  font-size: 12px;
  text-align: left;

  .post-category-list {
    display: inline-block;
    margin: 0;
    padding: 3px;
  }

搜索功能

到根目录安装 hexo-generator-searchdb

$ npm install hexo-generator-searchdb --save
